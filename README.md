# Hex Calc

A simple VS Code extension that opens a hex calculator in a webview panel.

## Features

- Input two hexadecimal numbers
- Select +, -, *, /
- Show the result in hexadecimal
- Copy the result

## Usage

1. Open Command Palette
2. Run `Hex Calc`
3. Enter A and B
4. Choose an operator
5. Copy the result if needed
